/boot - Загрузчик системы
/bin - Исполняемые файлы
/sbin - Бинарники тоже, для root обычно
/usr - Пользовательские аппы
/etc - Конфиги
/home - Домашняя папка 
/root - Домашняя папка root'а
/dev - Девайсы
/media - Монтирование внешних носителей
/mnt - Временные монтирования
/var - Динамические файлы, логи, кэш
/lib - Библиотеки
/lost+found - Восстановленные файлы после сбоев
/proc - Файлы с инфой о процессах и ОС
/run - Файлы состояния приложений
/srv - Файлы сервисов
/tmp - Временные файлы
/opt - Дополнительные необязательные аппы
